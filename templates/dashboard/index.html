Consumer Behavior and Shopping Habits:
A Deep Dive into Customer Choices

The project involves developing a Web Application using the Django framework. The application allows users to input various demographic and behavioral data, which is then processed by a machine learning model to predict the likelihood of High or Low purchasing behavior.

The models we trained are Naive Bayes, Support Vector Classifier(SVM), Random Forest Classifier, Logistic Regression, K-nearest neighbours(KNN), Decision Tree Regression, Gradient Boosting, and Ada Boost.
